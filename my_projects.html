College Marketplace

The College Marketplace project is a platform for students and staff to buy, sell, and exchange goods within the campus community. It features user authentication, categorized product listings, and in-app messaging for secure communication. The platform promotes sustainability by encouraging the reuse of items like books, electronics, and furniture. Built with a responsive and user-friendly design, it ensures seamless access and interaction. This project fosters a convenient and trusted campus economy, making transactions easy and efficient for all users.

CardioVision

The Cardiovascular Disease Detection project uses deep learning to analyze ECG images and classify heart conditions. Users can upload ECG data via an intuitive platform, which processes it using CNN for image analysis and KNN for classification. The system provides fast, accurate, and accessible diagnoses to support early detection of cardiovascular diseases. By addressing the challenge of limited access to cardiologists, the project offers a convenient and reliable healthcare solution.
